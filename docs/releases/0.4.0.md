# WorkKnowlage 0.4.0

**Release date:** 2026-05-15
**Stage:** Right Sidebar Wiki Associations

This release packages the current local-first desktop build with the right sidebar upgraded into a Properties / Wiki context panel. It also includes the accumulated editor, search, share, export, and project documentation hardening that was already present in the workspace before this release checkpoint.

## Highlights

- Right sidebar now has Properties and Wiki tabs, keeping document attributes separate from knowledge relations.
- Wiki tab shows an association badge and separates explicit references, related topics, and original-text evidence.
- Short exact phrase and key-sentence matches are derived as text evidence without lowering semantic related-topic thresholds.
- Clicking text evidence opens the target document block with fallback text for focus/highlight recovery.
- Workspace search, editor focus, share rendering, rich table behavior, and requirements documentation have broader regression coverage.
