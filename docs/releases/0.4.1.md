# WorkKnowlage 0.4.1

**Release date:** 2026-05-15
**Stage:** Right Sidebar Wiki Evidence Aggregation

This release refines the 0.4.0 right sidebar Wiki panel so users see document-centered associations instead of separate related-topic and original-text lists.

## Highlights

- Wiki tab now shows `关联文档` as the derived relationship layer below explicit references.
- Each associated document aggregates `主题相似`, `局部相似`, and `原文命中` evidence under one target document.
- Associated document cards show evidence badges and compact counts such as `1 处相似 · 1 条线索`.
- Hovering an associated document reveals the underlying similarity evidence and original-text evidence.
- Clicking evidence opens the target document block with fallback text for focus/highlight recovery.
- PRD, SPEC, and project memory now describe the association model as document-centered evidence aggregation.
