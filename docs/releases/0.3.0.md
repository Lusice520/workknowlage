# WorkKnowlage 0.3.0

**Release date:** 2026-05-07
**Stage:** P1 Complete

This release closes the full P1 roadmap tranche. The app/session/runtime boundaries from `0.2.0` remain intact, and the remaining large frontend refactors are now finished enough for a more stable long-use local build.

## Highlights

- `SharedBlockNoteSurface` is now a composition shell instead of a 1500+ line mixed-responsibility editor surface.
- Sidebar association derivation moved out of `RightSidebar` render flow and into `useSidebarAssociations`, with cache and invalidation guardrails.
- Sidebar behavior tests now render from prepared semantic association state, which makes the UI boundary explicit and easier to maintain.
- P1 contract coverage now includes app boundaries, editor composition boundaries, and sidebar data-layer boundaries.
