# WorkKnowlage 0.4.2

**Release date:** 2026-05-15
**Stage:** Right Sidebar Original Evidence Recall Fix

This release fixes the incomplete 0.4.1 sidebar association behavior for short original-text evidence in BlockNote list items.

## Highlights

- Original-text evidence now includes BlockNote `numberedListItem` and `checkListItem` content.
- Short list-item matches such as `LC 点位是否正确`, `设备动作是否正常`, and `联锁是否生效` are recalled as original evidence instead of being dropped.
- Short list-item evidence does not get promoted into semantic related-topic evidence by itself.
- Clicking an original evidence row now passes `highlightQuery` as the exact matched text, so the editor highlights the intended source phrase instead of relying on a longer snippet.
- Real local-data verification for `测试文档` now returns two associated documents: `问题清单宣贯稿内容` and `无标题文档`.
