# Project Memory

这是 WorkKnowlage 项目级 memory 层，属于 Lusice-Codex 自进化系统的一部分。

## Purpose

Project memory 记录只对 WorkKnowlage 有长期价值的经验，例如：

- 已踩过的工程坑。
- 产品和 UI 判断偏差。
- 架构边界。
- 测试和验证经验。
- 分享、编辑器、搜索、数据库等模块的 durable lessons。

## Files

- `project-lessons.md`：WorkKnowlage-specific lessons。

## Rule

如果经验是 WorkKnowlage-specific，写入这里；如果经验是 Lusice 和 Codex 的通用协作偏好，写入全局 memory。
