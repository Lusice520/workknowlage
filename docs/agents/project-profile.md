# Project Profile

WorkKnowlage 是一个 local-first macOS knowledge base application，技术栈是 Electron、React、BlockNote 和 SQLite。

## Product Positioning

产品定位接近 Obsidian-like local knowledge workflow tool，但更强调结构化编辑能力，以及未来 AI-assisted knowledge capabilities。

## Current Product Themes

- Local document management 和 tree navigation。
- Block-based editing，以及 advanced table behavior。
- Full-text search 和 search-result grouping。
- Tags、favorites、bidirectional links、sidebar associations。
- Export、sharing、local-first persistence。
- 未来方向包括 graph view、document templates、AI summarization、retrieval、Q&A。

## Important Product Biases

- 优先 local ownership 和 local-first behavior。
- 保持 practical、workflow-oriented，不做 marketing-style UI。
- Editor reliability、search quality、data safety 是核心质量。
- 避免把应用做成 generic SaaS dashboard 的感觉。
