# App Icon Design

**Goal**

为 `WorkKnowlage` 设计一枚可用于 macOS 应用图标、Dock 与 DMG 的正式图标。

**Chosen Direction**

- 图形语义：书页折角 + 三节点连线
- 品牌气质：极简线性
- 使用场景：应用图标、Finder、Dock、DMG

**Visual Spec**

- 底座：圆角方形，浅灰白底，轻微蓝灰边界，适配 macOS 浅色环境
- 主体：一张带折角的书页，用清晰蓝色线条勾勒
- 语义层：书页内用 3 个节点和 2 段连线表达“结构化知识”
- 强调色：延续当前产品的亮蓝色系统
- 风格原则：不使用厚重渐变，不使用复杂纹理，不做拟物阴影

**Small-Size Strategy**

- 保留书页轮廓和节点关系作为核心识别
- 线宽相对加粗，避免缩小时节点与连线消失
- 折角和节点位置保持清晰优先，不追求复杂细节

**Asset Pipeline**

- 主稿：`SVG`
- 基础位图：`1024x1024 PNG`
- macOS 图标：`ICNS`
- 安装包：`DMG`
